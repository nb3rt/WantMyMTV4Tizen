# WantMyMTV4Tizen
Samsung TV Tizen 3.0 compatible port of wonderful WantMyMTV.xyz / MTVReWind

for the original one and only MTVReWind head to https://wantmymtv.xyz/
and support the creator here https://buymeacoffee.com/flexasaurusrex

Drop me a line if you like it - it should get better in time and turn into actual Tizen TV app... 
right now it's just a webview benefitting from remote.

up / down - channels
left / right - previous / next
ok - pause / play
long ok - add to Favourites channel.

Have fun :)
